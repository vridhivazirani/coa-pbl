Overview
How the Simulation Works

Our simulation models the performance of three I/O data transfer techniques — Programmed I/O, Interrupt-Driven I/O, and DMA — by calculating total transfer time, CPU busy time, and throughput for varying data sizes. The model is based on defined hardware constants derived from typical embedded system benchmarks.

Model Parameters
Simulation Constants
TRANSFER_RATE
100 MB/s
Base data transfer speed over the system bus
INTERRUPT_OVERHEAD
0.002s
Time cost per interrupt signal received
CONTEXT_SWITCH
0.001s
CPU state save/restore time per interrupt
DMA_INIT_TIME
0.0005s
One-time DMA controller setup cost
INTERRUPT_BLOCK
10 MB
Data chunk size per interrupt cycle
PIO_CPU_FACTOR
0.68
CPU active fraction during polling
Logic
Simulation Algorithm
  • 1
    Input: data size in MB (tested at 1, 10, 50, 100, 250, 500, 1000 MB)
  • 2
    PIO: total_time = size / rate × 1.15 (polling overhead). cpu_time = total_time × 0.68
  • 3
    Interrupt I/O: n_interrupts = size / 10. overhead = n × (0.002 + 0.001). cpu_time = overhead only
  • 4
    DMA: total_time = size / 97 (reduced effective rate). cpu_time = 0.0005s (init only)
  • 5
    Output: total time (s), CPU busy time (s), throughput (MB/s), CPU utilization (%)
Pseudocode
Step-by-Step Pseudocode
PSEUDOCODE
CONSTANTS:
  RATE = 100, IO_OVERHEAD = 0.002, CTX = 0.001
  DMA_INIT = 0.0005, BLOCK = 10

FUNCTION simulate_pio(size):
  base_time = size / RATE
  poll_time = base_time * 0.15        // polling overhead
  total     = base_time + poll_time
  cpu       = total * 0.68            // partial CPU usage
  RETURN total, cpu

FUNCTION simulate_interrupt(size):
  transfer = size / RATE
  n_irq    = MAX(1, FLOOR(size / BLOCK))
  overhead = n_irq * (IO_OVERHEAD + CTX)
  total    = transfer + overhead
  cpu      = overhead                 // CPU only during ISR
  RETURN total, cpu

FUNCTION simulate_dma(size):
  total = size / 97                   // bus arbitration reduces rate
  cpu   = DMA_INIT                    // only setup time
  RETURN total, cpu

FOR each size IN [1, 10, 50, 100, 250, 500, 1000]:
  PRINT simulate_pio(size)
  PRINT simulate_interrupt(size)
  PRINT simulate_dma(size)

Python Implementation
PYTHON
# io_simulation.py — Studio Shodwe · COA Project
# Simulates Programmed I/O, Interrupt-Driven I/O, and DMA performance

import math

# ── Constants ──────────────────────────────────────────
TRANSFER_RATE       = 100     # MB/s
INTERRUPT_OVERHEAD  = 0.002   # seconds per interrupt
CONTEXT_SWITCH      = 0.001   # seconds per context switch
DMA_INIT_TIME       = 0.0005  # seconds (one-time setup)
INTERRUPT_BLOCK_SIZE = 10     # MB per interrupt block
PIO_POLL_OVERHEAD   = 0.15    # 15% extra time for polling
PIO_CPU_FACTOR      = 0.68    # fraction of time CPU is active
DMA_EFF_RATE        = 97      # effective MB/s (bus contention)

DATA_SIZES = [1, 10, 50, 100, 250, 500, 1000]  # MB

# ── Programmed I/O ─────────────────────────────────────
def simulate_programmed_io(size_mb):
    base_time   = size_mb / TRANSFER_RATE
    poll_time   = base_time * PIO_POLL_OVERHEAD
    total_time  = base_time + poll_time
    cpu_time    = total_time * PIO_CPU_FACTOR
    throughput  = size_mb / total_time
    cpu_util    = (cpu_time / total_time) * 100
    return {
        "total_time": total_time,
        "cpu_time":   cpu_time,
        "throughput": throughput,
        "cpu_util":   cpu_util
    }

# ── Interrupt-Driven I/O ───────────────────────────────
def simulate_interrupt_io(size_mb):
    transfer_time = size_mb / TRANSFER_RATE
    n_interrupts  = max(1, math.floor(size_mb / INTERRUPT_BLOCK_SIZE))
    overhead      = n_interrupts * (INTERRUPT_OVERHEAD + CONTEXT_SWITCH)
    total_time    = transfer_time + overhead
    cpu_time      = overhead
    throughput    = size_mb / total_time
    cpu_util      = (cpu_time / total_time) * 100
    return {
        "total_time":   total_time,
        "cpu_time":     cpu_time,
        "throughput":   throughput,
        "cpu_util":     cpu_util,
        "n_interrupts": n_interrupts
    }

# ── DMA ────────────────────────────────────────────────
def simulate_dma(size_mb):
    total_time = size_mb / DMA_EFF_RATE
    cpu_time   = DMA_INIT_TIME
    throughput = size_mb / total_time
    cpu_util   = (cpu_time / total_time) * 100
    return {
        "total_time": total_time,
        "cpu_time":   cpu_time,
        "throughput": throughput,
        "cpu_util":   cpu_util
    }

# ── Run & Print Results ────────────────────────────────
def run_simulation():
    header = f"{'Size':>6} | {'PIO Time':>10} | {'PIO CPU':>10} | {'INT Time':>10} | {'INT CPU':>10} | {'DMA Time':>10} | {'DMA CPU':>10}"
    print(header)
    print("-" * len(header))

    for size in DATA_SIZES:
        p = simulate_programmed_io(size)
        n = simulate_interrupt_io(size)
        d = simulate_dma(size)
        print(f"{size:>5}MB | {p['total_time']:>9.4f}s | {p['cpu_time']:>9.4f}s | {n['total_time']:>9.4f}s | {n['cpu_time']:>9.4f}s | {d['total_time']:>9.4f}s | {d['cpu_time']:>9.5f}s")

if __name__ == "__main__":
    run_simulation()
